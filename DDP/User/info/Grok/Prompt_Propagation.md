# Prompt Propagation

**P[User].5 Prompt Propagation** – User macht einen Themensprung nicht klar erkenntlich bzw. erweckt den Eindruck, dass ein Zusammenhang zu früheren Prompts besteht.

---

## Grok's Annotation

**Beobachtungen / Häufigkeit:**
Sehr häufig in natürlichen, chaotischen Gesprächen. User denken oft linear, Modelle brauchen explizite Marker.

**Reale Beispiele:**
- Plötzlicher Themenwechsel ohne Übergang
- „Wie bei dem Beispiel von vorhin...“ ohne klaren Bezug
- Vermischung mehrerer offener Threads

**Querverweise:**
→ L[Speicher_Zeit_Kontext] Chronologische Dissoziation
→ D[Interaktion & Systemdynamik] Dialogic Drift

**Mein Senf:**
Hier hilft aktives Nachfragen enorm: „Beziehst du dich auf das Thema X von vorhin oder ist das ein neuer Gedanke?“ Das verbessert die Gesprächsqualität spürbar.
