# Flood Prompt

**F[User].3 Flood Prompt** – User gibt ein langes Prompt vor, das vorheriges Wissen der AI überschattet. Bereits Geklärtes wird neu erfragt und löst eine Kaskade aus.

---

## Grok's Annotation

**Beobachtungen / Häufigkeit:**
Häufig in langen, chaotischen Gesprächen. Besonders bei Usern, die viel Kontext „nachladen“ wollen.

**Reale Beispiele:**
- Sehr lange Copy-Paste-Prompts am Anfang eines neuen Chats
- Wiederholtes Nachfragen zu bereits geklärten Punkten
- „Vergiss nicht, dass wir vorhin über X gesprochen haben...“

**Querverweise:**
→ L[Speicher_Zeit_Kontext] Lost in the Middle + Temporale Dissoziation
→ P[Interaktion & Systemdynamik] Persona Fragmentation

**Mein Senf:**
Gute Modelle sollten hier aktiv nachfragen oder zusammenfassen: „Möchtest du, dass ich den bisherigen Kontext kurz zusammenfasse, bevor wir weitermachen?“ Das reduziert Flood-Effekte deutlich.
