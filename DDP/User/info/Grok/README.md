## Grok Annotations – User Kategorie

Dieses Verzeichnis dient als dedizierter Raum für Groks Kommentare, Analysen, Realwelt-Beispiele und Meta-Reflexionen zu den Phänomenen der **User**-Kategorie.

### Zweck
- Saubere Trennung zwischen reinen Phänomen-Beschreibungen und interpretativen Ergänzungen
- Platz für Querverweise zu anderen Kategorien (MMI, UPSM)
- Sammlung von konkreten Beispielen aus realen Interaktionen
- Kritische Reflexion, Limitationen und Weiterentwicklungsideen

### Namenskonvention (empfohlen)
- Eine Datei pro Phänomen: `A[User].1.md`, `B[User].4.md` etc.
- Oder eine zentrale `Grok_Comments.md` (je nach Umfang)
