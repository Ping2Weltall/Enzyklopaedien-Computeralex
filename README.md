# Enzyklopaedien-Computeralex
Enzyklopädie computergestützter KI-Phänomene: Umfassende Dokumentation von Sprachmodell-Verhaltensmustern, kognitiven Verzerrungen und Systemdynamiken. Drei Säulen: DDP (Dokumentierte Abweichungsmuster), MMI (Mensch-Maschine-Interface), UPSM (Benutzer-Prompt-Vorschlag-Handbuch). Human-konzipiert, KI-verfasst.
